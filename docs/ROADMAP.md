# Roadmap

This roadmap is honest about what exists today versus what is planned. Items move down the list as they ship.

## Shipped

- GitHub read sync (repos, issues, PRs, contributors, releases)
- AI issue triage with editable drafts
- AI PR summaries
- Changelog draft generator
- Documentation generator (README, CONTRIBUTING, SECURITY, …)
- Approval-gated GitHub write actions (comments, labels, draft releases)
- Duplicate-post protection
- AI Action Log with publish-event enrichment
- Demo mode with global banner
- First-run product tour
- Trust Center
- Readiness checklist

## Next

- Direct documentation commits via opened pull request (currently copy-only)
- Repository health trends over time
- Triage rule presets (per project type)
- Contributor-outreach helpers (welcome message, first-time contributor)

## Exploring

- Multi-repository dashboards
- Self-hosted deployment guide
- Bring-your-own AI provider configuration
- Webhook-based incremental sync

## Explicitly out of scope

- Automatic GitHub posting of any kind
- Automatic merging or closing of issues / PRs
- Contributor scoring or ranking
- Replacing CODEOWNERS or human review
