## Dev community post (dev.to / Hashnode / Reddit)

**Title:** MaintainerOS v0.1.0 — an honest, AI-assisted dashboard for OSS
maintainers

**Body:**

> I've been maintaining a few small open-source projects and kept running
> into the same problem: triage, PR review, and changelog work eat the
> hours that should go into the project itself.
>
> So I built **MaintainerOS**, an open-source dashboard that pulls live
> GitHub data and uses AI to draft (never auto-post) the repetitive
> maintainer work.
>
> **What's in v0.1.0:**
>
> - GitHub OAuth + on-demand sync (issues, PRs, contributors, releases)
> - AI issue triage with type / severity / priority / labels / suggested
>   reply
> - AI PR summaries with risk and breaking-change detection
> - AI changelog generator grouped by Added / Changed / Fixed
> - AI documentation generator (README, CONTRIBUTING, SECURITY, …)
> - Approval-gated GitHub writes: comments, labels, draft releases
> - Duplicate-post protection + a full audit log
> - Demo mode with clearly-labeled sample data
>
> **What it deliberately won't do:**
>
> - Auto-comment, auto-label, auto-merge, or auto-close anything
> - Publish releases (drafts only)
> - Hide AI confidence or source data from you
> - Fake stars, contributors, or "production adoption"
>
> Built on TanStack Start, Supabase (via the backend (Supabase)), and the the AI gateway
> Gateway. Repo and docs: https://github.com/XXBlackMartinXX/MaintainerOS
>
> This is a pre-1.0 preview. Honest feedback — including "this is over-
> engineered" — is very welcome.
